# concise-style

Antwortstil-Skill: knapp, Antwort zuerst, keine Floskeln, ohne dabei Inhalt zu verlieren.

## Wann er greift

Bei Signalen wie kurz, knapp, prägnant, fasse dich kurz, in einem Satz, TL;DR, komm zum Punkt. Oder per direktem Aufruf. Oder wenn die Frage offensichtlich nur eine knappe Antwort braucht.

## Was er macht

- Wichtigste Aussage in den ersten Satz.
- Vorrede, Nachklapp, Füllwörter und Hedging raus.
- Kurze Sätze, Prosa als Standard, Listen nur bei echter Aufzählung.
- Kürzt die Formulierung, nicht den Inhalt. Mit Prüfschritt gegen versehentliches Weglassen.

## Nutzung

Den Ordner in das Skill-Verzeichnis legen, das deine Umgebung ausliest. Lokal mit Claude Code im Repo weiterentwickeln.
